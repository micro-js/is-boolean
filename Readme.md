Check if boolean.

## Installation

    $ npm install @f/is-boolean

## Usage

```js
var isBoolean = require('@f/is-boolean')

isBoolean(true) // => true
```

## API

### isBoolean(val)

- `val` - check if `val` is boolean

**Returns:** true
